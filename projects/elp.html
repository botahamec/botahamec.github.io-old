Elp

Elp is a git helper to make some repetitive tasks much easier to do. Elp only has three commands, so most of what you to do can be done with little effort whatsoever.

Usage

Elp has a grand total of three commands, all of which require that you have a blank repository to start

elp start -u

This command creates a local repository and pushes it to the remote repository linked. For example: https://www.github.com/Botahamec/elp.git

elp push [TITLE] -m [MESSAGE]

This command adds all the files in the current folder to the local repository and makes a commit with the given title and optional message. It then pushes the commit to the remote wrepository

elp pull

Pulls from the remote repository. Simple enough.

Building

In order to build this, you'll need to have Cargo (the Rust compiler) and Git pre-installed.

  1. Clone the repository (git clone https://github.com/Botahamec/elp.git)
  2. Run the installer script. On Unix systems, it's make_linux.sh and on Windows it's make_win.cmd
  3. Verify that it installed correctly by typing "elp -V" in a terminal
